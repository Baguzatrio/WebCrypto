# Analisis Harga Cryptocurrency Berbasis Data Historis (Berbasis Web)

## Latar Belakang
Harga cryptocurrency bersifat fluktuatif dan sulit dianalisis hanya dari
harga saat ini. Diperlukan analisis data historis untuk memahami pola
pergerakan harga, tren, dan volatilitas aset kripto secara lebih objektif.

Project ini dikembangkan dalam bentuk **aplikasi web** agar hasil analisis
dapat diakses secara interaktif dan mudah dipahami oleh pengguna.

## Aplikasi Web
- URL: https://ifupnvjtcrypto.my.id
- Fitur utama:
  - Menampilkan data harga cryptocurrency
  - Visualisasi grafik pergerakan harga
  - Dashboard analisis data historis

## Dataset
- Sumber data: Public API cryptocurrency
- Jenis data: Time series harga aset kripto

## Pengumpulan dan Pengolahan Data
- Pengambilan data harga melalui REST API
- Penyimpanan data historis menggunakan MySQL
- Pengolahan data untuk analisis tren dan pergerakan harga

## Analisis Data
- Analisis pergerakan harga berdasarkan data historis
- Identifikasi tren naik, turun, dan fluktuasi harga

## Visualisasi
- Grafik pergerakan harga aset kripto
- Dashboard berbasis web untuk menampilkan hasil analisis

## Kesimpulan & Insight
Analisis data historis menunjukkan bahwa pergerakan harga cryptocurrency
memiliki pola fluktuasi yang signifikan dalam periode tertentu. Penyajian
hasil analisis melalui aplikasi web membantu pengguna memahami tren harga
secara lebih intuitif dibandingkan hanya melihat harga real-time.
